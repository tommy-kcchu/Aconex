Running without phone number file, the program will prompt user to get the phone number from STDIN:

USAGE: java -jar AconexPhoneChallenge.jar  -d <<DIRECTORY_PATH>>

Running with a dictionary and phone number file, you can execute the following command 
where <<DIRECTORY_PATH>> is your dictionary path and <<PHONE_NUMBER_PATH>> is the location of your phone number list 

USAGE: java -jar AconexPhoneChallenge.jar  -d <<DIRECTORY_PATH>> <<PHONE_NUMBER_PATH>>
